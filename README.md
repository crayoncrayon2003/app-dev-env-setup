## 📝 書籍概要

本書は、アプリケーション開発を始める方に向けた入門書です。**WSL2・Ubuntu・Python・Node.js・Docker・VS Code** の環境構築手順を、スクリーンショット付きで丁寧に解説しています。

「開発環境を構築したいが、何から始めればいいか分からない」という初心者が、最終的にDockerを使ったサンプルアプリを動かせる状態を目指します。

### 対象読者

- アプリケーション開発の経験がない方
- 将来、自作したアプリケーションをクラウドや組み込み機器で動作させたいと考えている方

## 🚀 サンプルアプリの使い方

### アプリの起動

```bash
cd 16_SampleApp
docker compose up -d
```

### アプリの停止

```bash
docker compose down
```
